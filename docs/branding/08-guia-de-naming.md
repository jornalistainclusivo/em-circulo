# Guia de Naming

## Visão Geral

Este documento estabelece os princípios para criação, avaliação e aprovação de nomes relacionados ao ecossistema da **Em Círculo**.

Seu objetivo é garantir consistência entre marca, produto, funcionalidades, serviços e futuras iniciativas, preservando a identidade verbal construída pela plataforma.

Este guia baseia-se nas boas práticas de branding recomendadas pela **ABA (Associação Brasileira de Anunciantes)** e pela **Bradda Branding**, adaptadas ao contexto de um produto digital voltado ao cuidado compartilhado.

---

# Objetivos

Todo nome criado para a Em Círculo deve:

* fortalecer a marca principal;
* ser facilmente compreendido em português brasileiro;
* ser memorável;
* ser pronunciado naturalmente;
* transmitir colaboração e confiança;
* favorecer a expansão futura do produto.

# Princípios de Naming

## 1. Clareza

O usuário deve compreender o significado do nome sem precisar de explicações.

### Preferimos

* Círculo
* Pessoas
* Convites
* Histórico
* Medicamentos

### Evitamos

* Hub
* Workspace
* Dashboard
* Care Center
* Workflow

---

## 2. Simplicidade

Os melhores nomes costumam ser curtos.

Preferencialmente:

* uma palavra;
* até três sílabas;
* leitura imediata.

Exemplos:

* Agenda
* Tarefas
* Avisos
* Arquivos

## 3. Naturalidade

Os nomes devem parecer parte do vocabulário cotidiano.

O usuário deve sentir que já conhece a palavra.

---

## 4. Humanização

Sempre priorizar nomes relacionados às pessoas.

A plataforma organiza relações.

Não processos.

## 5. Escalabilidade

Os nomes devem continuar fazendo sentido quando o produto crescer.

Evitar nomes excessivamente específicos.

# Personalidade Verbal

Todo novo nome deve transmitir:

* acolhimento;
* colaboração;
* simplicidade;
* confiança;
* organização.

Nunca deve transmitir:

* burocracia;
* complexidade;
* frieza;
* autoridade;
* excesso de tecnologia.

# Critérios de Avaliação

Cada proposta deve ser avaliada segundo os seguintes critérios.

| Critério       | Pergunta                                  |
| -------------- | ----------------------------------------- |
| Clareza        | O significado é imediato?                 |
| Pronúncia      | É fácil de falar?                         |
| Escrita        | É fácil de escrever?                      |
| Memorização    | É fácil de lembrar?                       |
| Diferenciação  | Evita confusão com concorrentes?          |
| Escalabilidade | Funciona para futuras expansões?          |
| Humanização    | Aproxima pessoas?                         |
| Coerência      | Está alinhado ao posicionamento da marca? |

Um nome somente deve ser aprovado quando atender satisfatoriamente a todos os critérios.

# Processo de Criação

## Etapa 1 — Definir o objetivo

Responder:

* O que será nomeado?
* Quem utilizará esse nome?
* Em qual contexto?

## Etapa 2 — Levantar o território semântico

Consultar o documento **Território Semântico**.

Selecionar palavras relacionadas aos conceitos centrais da marca.

Exemplos:

* círculo
* rede
* presença
* apoio
* cuidado
* confiança
* colaboração
* organização
* conexão

---

## Etapa 3 — Gerar alternativas

Produzir uma lista ampla de possibilidades.

Nesta etapa, quantidade é mais importante que qualidade.

Misturar:

* substantivos;
* metáforas;
* palavras compostas;
* expressões;
* construções inéditas.

---

## Etapa 4 — Filtrar

Eliminar nomes que:

* sejam difíceis de pronunciar;
* possuam grafia ambígua;
* dependam de explicação;
* sejam excessivamente técnicos;
* limitem futuras evoluções.

---

## Etapa 5 — Validação

Antes da aprovação final, verificar:

* disponibilidade de domínio;
* disponibilidade em lojas de aplicativos;
* disponibilidade de marca (INPI);
* disponibilidade em redes sociais;
* conflitos fonéticos ou semânticos.

Nenhum nome deve ser aprovado apenas por disponibilidade jurídica.

A estratégia da marca continua sendo o principal critério.

# Tipos de Nome

## Descritivos

Explicam diretamente a função.

Exemplos:

* Agenda
* Histórico
* Convites

Vantagem:

Alta compreensão.

Desvantagem:

Menor diferenciação.

---

## Evocativos

Criam uma associação emocional.

Exemplos:

* Em Círculo
* Laço
* Aldeia

Vantagem:

Maior construção de marca.

Desvantagem:

Exigem maior investimento em posicionamento.

## Metafóricos

Representam um conceito.

Devem ser utilizados apenas quando reforçarem claramente o território da marca.

Evitar nomes associados a:

* hospital;
* doença;
* enfermidade;
* sofrimento;
* dependência;
* burocracia;
* vigilância;
* monitoramento;

* anglicismos desnecessários;
* siglas;
* abreviações pouco intuitivas;
* grafias artificiais;
* números;
* caracteres especiais.

# Checklist de Aprovação

Antes de oficializar um novo nome, confirme:

* O nome é fácil de pronunciar?
* É fácil de memorizar?
* É fácil de escrever?
* Faz sentido para pessoas de diferentes idades?
* Está alinhado ao território semântico da marca?
* Reforça colaboração e rede de apoio?
* Continua fazendo sentido daqui a dez anos?
* Possui disponibilidade jurídica e digital?

Se alguma resposta for negativa, o processo deve retornar às etapas anteriores.

# Governança

Toda proposta de nome deve ser analisada conjuntamente por representantes de:

* Produto;
* Design;
* Branding;
* UX Writing;
* Marketing;
* Jurídico (registro de marca).

A decisão final deve equilibrar estratégia, experiência do usuário e viabilidade de proteção da marca.

# Resumo

O naming da **Em Círculo** deve refletir a mesma simplicidade e humanidade que orientam todo o produto.

Os nomes não existem apenas para identificar funcionalidades.

Eles constroem significado.

Cada palavra escolhida deve reforçar a promessa da marca:

> **A tecnologia aproxima pessoas.**
>
> **A linguagem fortalece vínculos.**
>
> **Os nomes ajudam a construir uma rede de apoio.**
